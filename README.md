# Green Autonomous Solar-Electric Cart System

> Production-Grade Autonomous Mobility Platform for Smart Campuses, Hospitals, Tourism Parks, Industrial Zones, and Rural Healthcare Centers.

## Features

### Passenger Application

- Google, OTP, Guest and Admin Authentication
- QR-Based Pickup System
- Real-Time GPS Tracking
- Autonomous Ride Management
- Voice Assistance (English & Tamil)
- Digital Payments (UPI, Razorpay, Google Pay, PhonePe and Paytm)
- Emergency Safety Controls
- Dark & Light Mode Support
- Live ETA Monitoring
- Push Notifications

### Autonomous Vehicle System

- GPS & GPS RTK Navigation
- LiDAR Mapping
- IMU Processing
- Ultrasonic Obstacle Detection
- Camera-Based Computer Vision
- AI-Based Route Optimization
- Autonomous Emergency Braking
- Collision Detection
- Sleep Mode Management
- Smart Charging Recommendations

### Admin Dashboard

- Multi-Cart Monitoring
- Fleet Management
- Live GPS Tracking
- Revenue Analytics
- Battery Monitoring
- Charging Statistics
- Payment Monitoring
- Emergency Management
- AI Prediction Reports
- Route Management

## Technology Stack

| Layer | Technology |
|-------|------------|
| Mobile App | Flutter |
| Dashboard | Flutter Web |
| Backend | NodeJS |
| AI Services | FastAPI |
| Database | PostgreSQL + Firebase |
| AI | TensorFlow + OpenCV |
| Navigation | ROS2 |
| Maps | Google Maps API |
| Communication | MQTT + WebSockets |
| Cloud | AWS + Google Cloud |
| Payments | Razorpay + UPI |
| Edge Computing | NVIDIA Jetson Nano |

## Deployment

### Flutter

```bash
flutter pub get

```

### Backend

```bash
npm install
npm run dev
```

### FastAPI

```bash
pip install -r requirements.txt
uvicorn app:app --reload
```

### ROS2

```bash
colcon build
source install/setup.bash
```
